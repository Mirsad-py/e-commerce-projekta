# E-commerce projekat

Ovo je frontend e-commerce projekat izrađen pomoću HTML-a, CSS-a i JavaScript-a. Projekat predstavlja jednostavnu online trgovinu sa više stranica, prilagođenim dizajnom i organizovanom strukturom fajlova.

## Opis projekta

Cilj projekta je prikaz izrade moderne i responzivne web stranice za online kupovinu. Aplikacija sadrži osnovne stranice koje su potrebne za jednostavan e-commerce sistem, uključujući pregled proizvoda, prijavu korisnika i proces naplate.

## Funkcionalnosti

- Prikaz proizvoda na posebnoj stranici
- Stranica za prijavu korisnika
- Stranica za checkout / naplatu
- Responzivan dizajn
- Organizovan CSS i JavaScript kod
- Korištenje slika i dodatnih resursa za bolji izgled stranice

## Tehnologije

U projektu su korištene sljedeće tehnologije:

- HTML5
- CSS3
- JavaScript

## Struktura projekta

Projekt je organizovan kroz više fajlova i foldera:

- `checkout.html` – stranica za naplatu
- `login.html` – stranica za prijavu
- `products.html` – stranica sa proizvodima
- `css/` – stilovi projekta
- `js/` – JavaScript funkcionalnosti
- `images/` – slike korištene u projektu

## Svrha projekta

Ovaj projekat je izrađen kao vježba i praktični zadatak s ciljem usavršavanja frontend web development vještina, rada sa Git/GitHub alatima, te organizacije web projekta kroz više povezanih stranica.

## Pokretanje projekta

Za pokretanje projekta dovoljno je:

1. Preuzeti ili klonirati repozitorij
2. Otvoriti projektni folder
3. Pokrenuti željeni HTML fajl u browseru

Na primjer:
- `products.html`
- `login.html`
- `checkout.html`
